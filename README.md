\# Smart Timetable Management System - Frontend



\## Project Description

This repository contains the frontend of the Smart Timetable Management System, developed using Vue.js 3 and Vite. The application provides role-based interfaces for students, teachers, and advisors, allowing users to view timetables and manage timetable change requests.



\## Technologies Used

\- Vue.js 3

\- Vite

\- Vue Router

\- JavaScript

\- HTML

\- CSS



\## Run Locally



```bash

cd frontend

npm install

npm run dev

```



The application runs at \*\*http://localhost:5173\*\*.



> \*\*Note:\*\* The frontend communicates with a backend API running at `http://localhost:5000/api`. Ensure the backend server is running for full functionality.



\## Project Structure



\- `src/router` – Vue Router configuration and role-based navigation.

\- `src/views` – Route-level pages.

\- `src/components` – Reusable Vue components.

\- `src/composables` – Shared reactive state (authentication).

\- `src/api` – API communication layer.
